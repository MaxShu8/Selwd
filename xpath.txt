для текста:

//span[text()="Расписание рейсов"] - полное совпадение текста
//span[contains(text(), "Расписание рейсов")] - содержание какого-либо текста в тексте
//span[contains(., "Расписание рейсов")] - вместо text() - точка "." - более коротко, но менее понятно.
//span[starts-with(text(), "Вход")] - ищет селекторы, кот. содержат текст, кот. начинаются на "Вход"

для содержания не только в тексте:
//span[contains(@class, "active")] - по содержанию атрибута в классе 
//span[contains(@class, "act")][contains(@class, "ive")] - несколько условий по содержанию в классе

для отрицания:
//span[not(contains(@class, "active"))] - поиск селектора, кот. не включает атрибут active в теге class

для элементов:

//span[@class="text"]//li[@class="text-text"]/following-sibling::li - находит след. элемент с тегом li после указанного
//span[@class="text"]//li[@class="text-text"]/preceding-sibling::li - находит предыд. элемент с тегом li после указанного (вместо последнего li можно указать звездочку "*")
//li[@class="one" or @class="two" or @class="three"] - найти сразу несколько элементов
//li[@class="one"]//h3[last()] - найти последний элемент
//li[@class="one"]//h3[last()-1] - найти предпоследний элемент

родительские и дочерние элементы:

//li[@class="text-text"]/parent::span - находит родительский элемент (только на один выше по дереву)
//li[@class="text-text"]/ancestor::ul - находит родительский/прародительский элемент (выше по дереву на любой уровень)
//span[@class="text"]/li - дочерний элемент li
//span[@class="text"]//li - поиск по всем вложенным элементам

атрибуты:

//span[@class="text"][@text="text-text"] - совпадение по двум атрибутам для более точного поиска
либо так //span[@class="text" and @text="text-text"] - "and" вместо закрывающей и открывающей скобки.

подняться на один или немколько элементов выше от дочернего:

//li[@class="text-text"]/.. - выше на один элемент (до родительского)
//li[@class="text-text"]/../.. - выше два элемента (до прародительского)

математическая логика:

//li[@price!=5] - price не равно 5-ти
//li[@price>5] - price больше 5-ти
//li[@price=5] - price равно 5-ти
